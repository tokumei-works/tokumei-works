# Compliance Rules

このキットは収益化を目指しますが、アカウント停止や成果否認を避けるため、次を守ります。

## 必須表示

収益ページの冒頭またはフッターに以下を表示します。

```text
このページには広告・アフィリエイトリンク・有料コンテンツへのリンクを含む場合があります。
```

## 禁止する運用

- 「必ず稼げる」「誰でも月○万円」などの断定
- 架空の体験談、架空の実績、未確認のレビュー
- 自己購入、家族や知人への購入依頼、クリック依頼
- ASPに登録していない媒体での広告掲載
- 案件条件に反するSNS掲載
- 自動DM、自動フォロー、自動いいね、自動コメント
- トレンド便乗の大量自動投稿
- 医療、金融、法律、転職、美容で根拠のない断定
- 商品画像、価格、口コミの無断利用

## 公開前チェック

- [ ] PR/広告表記がある
- [ ] 誇大表現がない
- [ ] 出典が必要な主張に根拠がある
- [ ] リンク先が正しい
- [ ] 返金や価格について誤認がない
- [ ] 著作権侵害がない

## SNSルール

SNSは送客補助です。初期は `dist/social_queue.csv` を使い、各SNSの公式予約投稿や手動投稿で公開します。API連携は公式API・公式権限・明示同意が整った場合だけ有効化します。
